# 📑 실시간 RAG 문서 분석 챗봇 사용 가이드 (User Guide)

안녕하세요! 본 서비스는 사용자가 업로드한 문서(PDF, 이미지, 오피스 파일 등)를 인공지능(AI)이 실시간으로 읽고 학습하여, 문서의 내용에 대해 무엇이든 답변해 주는 **RAG(검색 증강 생성) 기반 문서 질답 챗봇**입니다.

어려운 회사 규정집, 긴 채용 공고(Job Description), 영수증, 공부해야 하는 전공 자료 등을 툭 던져서 쉽고 빠르게 원하는 정보만 쏙쏙 뽑아내 보세요!

---

## 🚀 아주 쉬운 3단계 이용 방법

### 1단계: 파일 업로드하기
* 메인 화면 왼쪽의 **[문서 업로드]** 박스 영역에 분석하고 싶은 파일을 마우스로 끌어다 놓으세요 (Drag & Drop).
* 또는 박스를 클릭하여 내 컴퓨터에 있는 파일을 직접 선택할 수도 있습니다.
* *지원 포맷: PDF, JPG, JPEG, PNG, BMP, TIFF, Word(.docx), Excel(.xlsx), PPT(.pptx), HTML*

### 2단계: AI 분석 대기 (약 3~5초 소요)
* 파일을 올리면 백그라운드에서 AI가 실시간으로 문서의 글자를 추출(OCR)합니다.
* 분석이 완료되면 초록색 불빛과 함께 **`🎉 문서 분석 및 OCR 완료!`** 메시지가 표시됩니다.
* 왼쪽 하단의 **`[📝 추출된 문서 원본 보기]`**를 누르면 AI가 읽어낸 텍스트 내용을 미리 확인해볼 수 있습니다.

### 3단계: 질문하고 답변 받기
* 문서 분석이 끝나면 화면에 **추천 질문 버튼(Chips)** 3개가 나타납니다.
  * **`📝 문서 핵심 내용을 3줄로 요약해 줘`**
  * **`🎯 주요 업무와 핵심 지원 자격 요건은 뭐야?`**
  * **`🎁 우대사항 및 복지 혜택 같은 특이사항이 있어?`**
* 추천 질문 버튼을 가볍게 클릭하거나, 하단의 입력창에 직접 문서와 관련된 어떤 질문이든 적어서 물어보세요!

---

## 💡 유용한 질문 팁
* **정확한 수치 물어보기:** *"이 채용 공고의 서류 접수 마감일이 정확히 언제야?"*
* **조건 검토하기:** *"지원 자격 요건 중에 관련 학과 제한이나 어학 성적이 필수 항목으로 지정되어 있어?"*
* **비교 분석하기:** *"우대사항 항목들만 모아서 표나 리스트로 가독성 있게 정리해 줘."*

---

## 🔒 개인정보 및 보안 안내 (안심하고 쓰세요!)
* 사용자가 업로드한 파일과 텍스트 정보는 질문에 실시간으로 답변하기 위한 **임시 컨텍스트(Context)로만 활용**됩니다.
* 서버나 데이터베이스에 파일이나 개인정보가 절대 **영구 저장되지 않으며**, 브라우저 창을 닫거나 새로고침하면 분석 데이터는 즉시 완벽하게 소멸합니다.
